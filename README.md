# Audio Player

## Vue3 + Vite

### App is accessed by the link [http://localhost:4173](http://localhost:4173), run the following commands to start up the server:

- npm i
- npm run build
- npm run preview

### Songs are stored in /src/assets/audios/ and should follow the naming convention: <kbd>band-name_album-name_song-name</kbd>.

### Album logos are stored in /src/assets/images/ and should follow the naming convention: <kbd>band-name_album-name</kbd>.
